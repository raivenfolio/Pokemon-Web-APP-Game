# Pokemon-Web-APP-Game 
It is a guess that pokemon inspired web-based game where it test players their knowledge of Generation 1 Pokemons by guessing their theirs from silhouettes!

## How it is Played 🕹️
The player will try to guess the pokemon based from its silhouette, an input of letter will be counted as a guess until the player exhaust their 'tries' or in this case the health! If the player guessed the pokemon wrong, loses a heart and the letter is added to the wrong guesses. Try captuing all the Generation 1 pokemons!
